# Demo Web Shop – Manual Testing Project

## Project Overview

This project demonstrates **manual testing of the Demo Web Shop e-commerce application**.
The goal of this project is to validate the core functionality of the application by creating test scenarios, writing detailed test cases, executing them, and reporting defects.

The testing focuses on major modules such as user registration, login, product search, cart management, and checkout.
